# 🌦️ Weather Forecast App (FastAPI Full Stack)

🚀 Real-time weather forecasting web application with **API integration**

---

## 🔥 Key Features

* 🌤️ Real-time weather data
* 📅 7-day forecast
* 📊 Interactive charts (Chart.js)
* 🌐 Fully responsive UI
* ☁️ Deployed on cloud

---

## 🛠️ Tech Stack

* FastAPI
* HTML
* CSS
* JavaScript
* Chart.js
* Weather API
* Render

---

## 🚀 Live Demo

👉 https://weather-app-fastapi-q7cc.onrender.com

## 💻 Installation

```bash
git clone https://github.com/Gubbalalavanya935/weather-app-fastapi
cd weather-app-fastapi
pip install -r requirements.txt
uvicorn main:app --reload
```
